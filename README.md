# Cronada — Cronometro de Natacion

La app ideal para entrenadores y profes de natacion.

## Funciones

- Cronometro multi-carril con inicio individual o simultaneo
- Gestion de alumnos y grupos
- Registro de tiempos por sesion (practica y competencia)
- Estadisticas, mejores marcas y graficos de progresion
- Exportacion a PDF y CSV
- Modo competencia con podio
- Soporte bilingue (Espanol / Ingles)
- Tema oscuro y claro

## Descarga

Disponible en Google Play (proximamente).
